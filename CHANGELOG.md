# 📜 Changelog

All notable changes to this project will be documented here.

## [2.0.0] - 2026-05-26

### Added
- 🔄 **Google Sheets auto-sync** — website pulls live data from your Google Sheet
- 🖼️ **Real phone image library** — 40+ models mapped to real product photos
- ⏱️ **Auto-refresh** — stock updates every 10 minutes automatically (configurable)
- 🔁 **Manual refresh button** in header
- 🚀 **GitHub Actions workflow** for auto-deployment
- 📁 **Modular file structure** with separate docs folder
- 📘 Comprehensive documentation:
  - `README.md` — overview
  - `docs/SETUP.md` — detailed setup guide
  - `docs/CUSTOMIZATION.md` — theming and customization
  - `docs/TROUBLESHOOTING.md` — common issues
  - `docs/WHATSAPP-AUTOMATION.md` — future automation roadmap

### Improved
- Smart phone name matching algorithm
- Better mobile responsive design
- Faster page load with lazy image loading
- Cleaner error states when data fails to load

## Versioning Format

We use [Semantic Versioning](https://semver.org/): `MAJOR.MINOR.PATCH`

- **MAJOR** — Breaking changes (e.g., new required config)
- **MINOR** — New features (backwards compatible)
- **PATCH** — Bug fixes
