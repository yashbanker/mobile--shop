# 🤝 Contributing

Thanks for your interest in improving this project! Here's how to contribute.

## 🐛 Reporting Bugs

1. Check [existing issues](../../issues) first — your bug might already be reported
2. Open a new issue with:
   - Clear title (e.g., "Search filter doesn't work for Samsung phones")
   - Steps to reproduce
   - Expected behavior
   - Actual behavior
   - Browser & OS info
   - Screenshots if applicable

## 💡 Suggesting Features

Open an issue with the **enhancement** label. Describe:
- The problem you're trying to solve
- Your proposed solution
- Why it would benefit other users

## 🔧 Pull Requests

1. **Fork** this repository
2. **Create a branch**: `git checkout -b feature/your-feature-name`
3. **Make your changes** to `index.html` or other files
4. **Test thoroughly**:
   - Open `index.html` locally
   - Verify all features still work
   - Test on mobile (responsive design)
5. **Commit**: `git commit -m "Added: brief description of change"`
6. **Push**: `git push origin feature/your-feature-name`
7. **Open a Pull Request** on GitHub
8. Describe what you changed and why

## 📋 Code Style

- Use 2-space indentation
- Keep functions small and focused
- Add comments for non-obvious logic
- Test in Chrome, Firefox, and mobile Safari
